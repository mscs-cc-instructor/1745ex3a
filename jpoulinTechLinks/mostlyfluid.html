Fix Network Issues (Windows 10)

Perform a Network Reset
Sometimes, you just need to perform a cold reset of the network configuration on your PC/laptop. This is how you do it. 1. Go to the Settings app (not Control Panel)
2. Go to Network and Internet
3. Click the Network Reset option under Status
4. Click Reset Now after reading and understanding the message displayed.


Manually installl/remove network driver(s) If you want to install/manage the driver of the network card you can use Device Manager through Control Panel. 1. Go to the Control Panel
2. Find and click on Device manager
3. Under Network adapters, find the adapter responsible for your internet
4. Right click the adapter and select properties
5. Select Driver
6. Perform your needed operation
